5. 设计概念 —「扬」不是一个字，是一座正在上升的几何建筑

左部扌 → 垂直主块 + 提笔走势
深蓝实块代表"立足之本"，向右上挑出的亮蓝三角代表"向上发力"

右部上日字框 → 开放几何框架
用轮廓线而非实心块表达"日"——开放的框架意味着视野和可能性

右部下展开形态 → 上升台阶 + K线柱
梯形底板代表稳步上升的台阶，内部竖线群呼应K线图/柱状图

金点右上角唯一暖色
全蓝体系中唯一的一点金——像一个签名、一个坐标原点

背景网格交易图表的潜意识引用
极淡的网格线让人联想到K线图/建筑设计图纸
