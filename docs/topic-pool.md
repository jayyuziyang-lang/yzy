## ⚡ 选题来源记录

### 方法一：热点驱动
- 监控：重大财报/政策发布/市场异动
- 响应周期：事件发生后48小时内启动评估
- 决策标准：事件影响力 + 用户关注度 + 可拆解深度

### 方法二：系列规划
- 预研4个系列：读懂宏观 / AI产业链 / 投资工具箱 / 市场机制
- 每个系列4-5期，建议按顺序发布
- 系列间隔：至少每2期插入1期非系列选题

### 方法三：用户反馈
- 跟踪：留言/私信高频问题
- 记录：常见误区/困惑
- 排期：每月至少1期来自用户反馈

### 方法四：编辑自选
- 来源：S-01跟踪的行业动态/研究报告
- 原则：必须有独特角度，不重复他人已做内容
