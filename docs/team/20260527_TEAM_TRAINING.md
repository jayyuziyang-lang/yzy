## 一、今天暴露的问题

### 问题1：图表停留在"能跑"而非"好看"

| 维度 | 之前 v3.0 | 问题表现 |
|------|----------|---------|
| 视觉 | matplotlib 默认输出 | 看起来像程序员跑出来的图，不像财经媒体制作的 |
| 背景 | 纯色 F8FAFC | 没有卡片层次感，平坦无层次 |
| 标题 | set_title 居中 13pt | 太小、不醒目 |
| 副标题 | 无 | 缺少时间/板块说明 |
| 来源 | 无 | 每张图都不知道数据从哪来的 |
| 脊线 | 保留左/下两条 | 多余线条干扰视线 |

**根本原因：** 只关注了代码逻辑正确（颜色、标签、数据），忽略了视觉设计。以为"功能正确=质量合格"。

**根本原因：** 图表清单和实际产出脱节，没有定期检查依赖文件。

## 二、v4.0 升级内容（必须掌握）

### ChartCard 布局引擎

所有图表现在必须遵循统一结构：

```
标题（14pt粗体，左对齐，#1E293B）
副标题（9pt灰色，#64748B，说明时间/内容）
┌─────────────────────────────────────┐
│  纯白图表面板（#FFFFFF）            │
│  · 零脊线（所有spine移除）          │
│  · 网格线仅水平/数值轴，置于数据下层 │
│  · 网格浅灰（#E2E8F0），透明度0.4    │
└─────────────────────────────────────┘
来源: ××××××（6.5pt灰色，#94A3B8）
```

### 设计理念：卡片式层次

```
图表面板背景（#F0F2F5 暖灰）   ← 容器层（制造卡片感）
  └─ 图表面板（#FFFFFF 纯白）   ← 数据层（干净清爽）
      └─ 网格线（#E2E8F0 浅灰）← 辅助层（置于数据下层）
```

### 配色规则（不变，继续执行）

| 场景 | 正/涨 | 负/跌 |
|------|-------|-------|
| A股/全球指数（红涨绿跌） | `#DC2626` 红 | `#16A34A` 绿 |
| 资金流向（红流入绿流出） | `#DC2626` 红 | `#16A34A` 绿 |
| 半导体/科技 | `#7C3AED` 紫色系 | — |
| 黄金/贵金属 | `#D4A017` 金色系 | — |

## 三、新的质量门禁标准

### 生产时必须执行的检查（不可跳过）

每次运行 `generate-charts.py` 后，确认：
1. ✅ **33/33 质量门禁通过**（v3.0是30/30，新增gold_oil_ratio后+3=33）
2. ✅ **每张SVG有 ChartCard 布局**（标题+副标题+来源脚注+纯白面板）
3. ✅ **背景是暖灰 #F0F2F5**（不是旧版 #F8FAFC，更不是纯白）
4. ✅ **所有spine已移除**（零脊线，干净）
5. ✅ **每张图表有数据来源**（不允许无来源脚注的图表通过）

### 历史图表清理

对于已存在的旧版SVG，如果发现无ChartCard布局，必须用 `generate-charts.py` 重新生成，不允许直接复用旧文件。

## 四、团队纪律

### 绝对禁止

| 禁止项 | 原因 |
|--------|------|
| ❌ 跳过 `generate-charts.py` 运行 | 图表不会自动更新，旧版SVG会留在目录 |
| ❌ 直接复制旧SVG文件当新图 | 旧SVG没有ChartCard布局，质量不合格 |
| ❌ 手动修改SVG替代重新生成 | 版本不一致，质量门禁无法自动校验 |
| ❌ 在mplstyle中设置font.family | 会覆盖 setup_font() 的中文字体配置 |
| ❌ plt.style.use() 在 setup_font() 之后执行 | 中文字体配置会被mplstyle覆盖 |

## 五、快速参考

### 新旧版本对照

| 场景 | 旧 v3.0（不合规） | 新 v4.0（合规） |
|------|------------------|----------------|
| 创建图表 | `fig, ax = plt.subplots()` + 手动style_ax | `cc = ChartCard(title, subtitle, source)` |
| 设置背景 | `fig.patch.set_facecolor(BRAND_BG)` | mplstyle v2.0 自动处理 |
| 图表面板 | 同背景色 | 自动纯白卡面板 |
| 标题 | `ax.set_title()` 居中 | ChartCard 自动左对齐 |
| 来源脚注 | 无 | ChartCard source= 参数必填 |
| 保存 | `fig.savefig()` | `cc.save(filename)` |

### 正常生产流程

```bash

# → 确认输出: "33/33 通过"
# → 确认视觉: 暖灰背景 + 纯白卡片 + 零脊线
```
