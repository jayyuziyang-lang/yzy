> 触发事件：伊朗冲突对黄金影响的分析出现系统性逻辑错误，同一错误出现在4处内容中

---

## 核心理念

信息准确不只是"数据正确"，更是"逻辑正确"。我们的产品有三层准确性：

| 层级 | 定义 | 我们做对了什么 | 做错了什么 |
|------|------|---------------|-----------|
| L1: 数据准确 | 每个数字正确可追溯 | 金价$4,502、PCE 3.8%、原油$120都正确 | — |
| L2: 归因准确 | 因果关系正确 | 霍尔木兹封锁→油价上涨→PCE上升正确 | — |
| L3: 分析准确 | 基于L1+L2的推论正确 | — | **停火→利好黄金的判断完全错误** |

**L3（分析准确）是这次系统性错误的真正短板。** 数据都对，归因都对，但推论错了。

## 第一章：来源权威性评级

每条信息来源在被引用前，必须进行权威性评级：

| 级别 | 标准 | 代表来源 | 可引用场景 |
|------|------|---------|-----------|
| **A级（最高）** | 一手官方数据、权威机构报告 | 各国央行官网、世界黄金协会、BIS、IMF、美联储、BLS、COMEX | 任何场景，无需额外验证 |
| **B级（高）** | 顶级投行研究、一流财经媒体 | Goldman Sachs、JPMorgan、BofA Global Research、Bloomberg、Reuters、华尔街见闻 | 可作为主要依据，建议交叉验证 |
| **C级（中）** | 主流财经媒体、行业研究 | 新浪财经、东方财富、财联社、搜狐财经、凤凰网、新华财经 | 仅引用数据事实，不引用分析观点 |
| **D级（低）** | 自媒体、非专业来源 | 个人公众号、非认证分析师、社交媒体 | **不可引用** |

### 市场分析观点的特殊规则

对于市场分析/预测类观点（非数据事实），**即使来自B级来源**，也需要：

1. **反向论证**：主动寻找反方向的观点（如果某投行看多，必须至少找到一家看空的观点作为对照）
2. **观点标识**：在文章中清晰标注"这是观点而非事实"（如"XX认为"、"XX指出"）
3. **观点来源**：分析观点必须有出处（谁说的、什么机构、什么报告）

## 第二章：数据交叉验证协议

### 2.1 强制性验证规则

| 数据类型 | 最低独立来源数 | 验证要求 |
|---------|---------------|---------|
| 市场价格（指数、金价、原油等） | 2个 | 两个独立数据源交叉确认 |
| 宏观数据（PCE/GDP/CPI/失业率） | 2个 | 官方数据（BLS/央行）+ 至少1个媒体报道 |
| 历史对比数据（"X年最大/首"） | 2个 | **必须**追溯原始数据序列，不可依赖二手来源 |
| 公司财务数据 | 1个 | 官方财报原文（SEC filing 或公司官网） |
| 央行储备/购金数据 | 2个 | IMF IFS + 各国央行官网/世界黄金协会 |
| 投行研报观点 | 1个 | 指明具体报告名称/日期 |

### 2.2 禁止使用的表述

以下表述在没有至少2个独立来源追踪原始数据序列的情况下，**严格禁止使用**：

```
❌ "创X年最大/新高"（除非追踪原始数据序列并标注区间）
❌ "史上第X次"（除非有完整历史数据验证）
❌ "X年以来首次"（同上）
❌ "市场普遍认为"（谁认为？拿出具体来源）
❌ "分析人士指出"（哪位分析人士？）
❌ "值得关注"（关注什么？为什么？）
❌ "后市可期"（空洞套话，永久禁用）
```

## 第三章：核心逻辑验证门禁（新增）

> **这是本次复盘新增的最重要的质量门禁。**
> 适用于所有涉及因果推演、市场分析、趋势判断的内容。任何"因为A所以B"的论断，必须经过以下验证。

### 3.1 三段式逻辑验证

每一条分析论断，必须通过三个层面的自问：

```
L1 数据检查：支撑论断的每个数据点正确吗？
    → 金价$4,502、PCE 3.8%、油价$120 ✓

L2 归因检查：数据之间的因果关系成立吗？
    → 霍尔木兹封锁→油价涨→PCE涨→Fed鹰派 ✓

L3 推理检查：从归因推导出的结论是否自洽？
    → 停火→油价降→PCE降→Fed宽松→利好黄金 ✗ ← 错的就在这里
    → 正确推理：停火→避险溢价消退+通胀根源移除非自动利好→综合偏利空 ✓
```

**L3 是最容易被忽略的环节。** 具体验证方法见下。

### 3.2 L3推理验证的具体方法

#### 方法A：方向性检验

对于任何"事件X→资产Y涨/跌"的论断，检查：

1. **该资产当前价格水平**：
   - 黄金在$4,500+，是不是已经反映了这个利好？（→ 是，旧模型说的"停火利好"已经被定价了）
   - 如果X发生，Y还有多少上行空间？
   
2. **主要矛盾 vs 次要矛盾**：
   - 列出所有传导渠道，标注哪个是主要哪个是次要
   - 验证主要矛盾的方向**不是**你想当然的方向

#### 方法B：历史类比检验

对于当前事件，找到最近的历史可比案例：

```
事件：伊朗冲突停火 → 黄金会怎样？
类比：俄乌冲突2022年3月谈判 → 金价3个月内从$2,070跌至$1,620
验证结果：方向偏空（与旧分析结论相反）
```

**注意：** 历史类比需要标注类似度（"高度类似/部分类似/参考价值有限"），不可过度依赖单一历史类比。

#### 方法C：双渠道/多因子分析

对于任何资产价格影响的判断，必须列出**至少两条独立传导渠道**，分别判断方向：

```
停火事件 → 黄金价格影响：
  渠道1：避险溢价消退 → 利空黄金 ✓
  渠道2：通胀压力减轻 → Fed鹰派放缓 → 原本认为利好
    但：通胀预期下降 → 实际利率可能走高 → 利空同样成立
  综合：两条都偏空 → 正确结论应为利空
```

**失败检测信号：** 如果一条渠道方向明显、另一条需要"微妙论证"才能得出同向结论——很可能你的推理有偏误。

### 3.3 价格位置自检

这是最简单的预防性检查。在做出任何趋势判断前，先回答：

> **该资产当前价格，处于历史什么位置？**

- 黄金$4,500+ → 历史最高区间（ATH附近）。这意味着可预见的"利好"大概率已被定价。
- 如果某个事件被广泛认为是"利好"，而资产已经在历史最高位——这个利好可能已经price in了。
- **反直觉的"利空在ATH"比"利好"更可能发生。**

```
自检规则：
- ATH附近的资产：对"利好"叙事高度警惕
- 底部区域的资产：对"利空"叙事高度警惕
- 不要在ATH附近说"还会大涨因为XX利好"
- 不要在底部说"还会大跌因为XX利空"
```

## 第四章：前置分析与后置核查流程

### 4.1 分析论断生产流程

```
┌─────────────────────────────────────────────────┐
│  步骤1: 搜集数据（L1验证）                      │
│  → 每个数据点标注来源URL+等级                   │
├─────────────────────────────────────────────────┤
│  步骤2: 因果推演（L2验证）                      │
│  → A是否真的导致B？是否存在第三条变量？         │
├─────────────────────────────────────────────────┤
│  步骤3: L3推理验证（新增 · 强制）              │
│  → 方向性检验 + 历史类比 + 多渠道分析          │
│  → 填写《分析论断审核表》（见第五章）           │
├─────────────────────────────────────────────────┤
│  步骤4: 反方观点检查                            │
│  → 至少找到一个与你结论相反的观点/来源          │
│  → 如果你的结论没有反面观点 → 重新审视         │
├─────────────────────────────────────────────────┤
│  步骤5: 终审放行                                 │
│  → Claude Code逐项核查L1/L2/L3                 │
│  → 每项标记为✅或❌，❌不通过不放行             │
└─────────────────────────────────────────────────┘
```

### 4.2 错误响应的快速反应协议

当用户指出或发现信息错误时：

1. **立即冻结**：相关文章在确认修正前停止访问（或标注"修正中"）
2. **追踪所有副本**：使用grep扫描所有文件中相同/相似表述
3. **根因分析**：属于L1/L2/L3哪一层级错误？为什么会发生？
4. **系统性修复**：不只是修正当前文章，还要修复所有受影响的内容
5. **流程注入**：这个错误暴露了什么流程漏洞？如何防止再犯？

## 第六章：本次错误复盘（作为反面教材永久存档）

### 错误描述
伊朗冲突停火对黄金的影响，在4处内容中被误判为"利好"。

### 为什么错了？
1. 只看了"通胀→加息"这一条传导链，忽略了**主要矛盾是避险溢价消退**
2. 在黄金ATH（$4,500+）位置谈"利好"，**位置自检缺失**
3. 没有做历史类比（俄乌冲突2022年停火谈判后的金价走势）
4. 没有找反方观点验证

### 正确分析
| 渠道 | 方向 | 强度 | 说明 |
|------|------|------|------|
| 避险溢价消退 | 利空 | 主要矛盾 | 金价从$2,600→$4,500+中避险溢价贡献显著，停火后将退出 |
| 通胀根源移除 | 利空 | 次要矛盾 | 通胀预期下降→实际利率可能走高，不是简单利好 |
| 综合 | 利空 | — | 预计回落至$4,300-4,400 |

### 教训
- ✅ 数据准确 ≠ 分析正确
- ✅ 市场在ATH时，"利好"往往是陷阱
- ✅ 因果链不能只推一条——必须推完整网络
- ✅ 反方观点不是威胁，是安全网
