# 扬说·深度 — 选题标准白皮书 v1.0

> **核心理念：** 选题是专题的灵魂。宁缺毋滥，每期必精。
> 更新: 2026-05-22

---

## 一、选题"三不碰"铁律

任何选题在进入评估前，先用这三条过滤。**触碰任何一条，直接淘汰。**

### ❌ 不碰百度能搜到的

如果用户百度一下就能看到清晰解释，这个选题没有价值。

| 淘汰例子 | 问题 |
|---------|------|
| "什么是市盈率" | 百度百科比我们写得更清楚 |
| "黄金为什么是避险资产" | 这是所有财经号的入门级选题 |
| "什么是量化交易" | 基础概念，没有"扬说"的独到角度 |

**通过标准：** 用户至少需要读5分钟才能理解的东西，而非30秒百度能解决的问题。

### ❌ 不碰别人做烂了的

如果全网有超过20篇同类文章，我们需要至少3个角度创新，否则不做。

| 淘汰例子 | 问题 |
|---------|------|
| "美联储加息影响" | 2022-2023年已经被写到泛滥 |
| "英伟达为什么涨" | 全网的财经号都在写 |
| "巴菲特的投资哲学" | 被写了几十年了 |

**唯一例外：** 我们发现了一个全新的切入角度，能讲出别人没讲过的东西。

### ❌ 不碰纯事实罗列

没有判断、没有分析、没有"为什么"的选题，不是深度专题。

| 淘汰例子 | 问题 |
|---------|------|
| "美股七巨头2026年财报汇总" | 纯数据搬运 |
| "A股历年走势回顾" | 没有分析框架 |
| "各国GDP排名变化" | 没有因果关系 |

**通过标准：** 读者看完会说"原来如此"，而不是"哦，知道了"。

## 二、选题五维评分模型

每个候选选题必须经过五维评分，**总分低于75分直接淘汰**。

### 评分维度

| 维度 | 权重 | 评分标准 | 满分 |
|------|------|---------|------|
| **1. 不可替代性** | 30% | 这个角度只有我们能讲/能讲好 | 30 |
| **2. 认知升级度** | 25% | 读者能获得多少新的认知 | 25 |
| **3. 时效锚点** | 15% | 为什么是现在讲？时效紧迫性 | 15 |
| **4. 专业纵深** | 20% | 能不能展示专业功底 | 20 |
| **5. 可传播性** | 10% | 用户愿不愿意转发/收藏 | 10 |

### 评分细则

#### 1. 不可替代性（30分）

| 分数 | 判断标准 | 举例 |
|------|---------|------|
| 25-30 | 全网几乎没有同类深度内容 | 《央行法修订：一部法律如何影响你的钱袋子》 |
| 15-24 | 有人写过但角度完全不同，或我们的深度远超对手 | 《A股政策市的底层逻辑》（对比普通科普） |
| 5-14 | 常见选题，但我们的解释方式/漫画可视化有独特价值 | 《美债收益率倒挂》（虽然很多人写，但我们用漫画讲因果链） |
| 0-4 | 百度/知乎/公众号已经有大量同质内容 | 直接淘汰 |

#### 2. 认知升级度（25分）

| 分数 | 读者看完后能…… |
|------|----------------|
| 20-25 | 建立一个新的分析框架，能用来分析其他问题 |
| 12-19 | 理解一个之前完全不懂的机制/逻辑链 |
| 5-11 | 知道一些新的事实和数据 |
| 0-4 | 没有获得新的认知 |

#### 3. 时效锚点（15分）

| 分数 | 判断标准 |
|------|---------|
| 12-15 | 正在发生的重大事件，用户急需理解（如突发政策/财报/黑天鹅） |
| 7-11 | 近期热点，但一周后关注度会下降 |
| 3-6 | 长期话题，任何时间讲都可以 |
| 0-2 | 过时话题，或者热点已过 |

#### 4. 专业纵深（20分）

| 分数 | 判断标准 |
|------|---------|
| 15-20 | 需要真正的专业功底（产业链拆解/机制分析/数据交叉验证） |
| 8-14 | 有一定专业深度，但勤奋的爱好者也能写 |
| 2-7 | 常识性内容 |
| 0-1 | 纯搬运/翻译 |

#### 5. 可传播性（10分）

| 分数 | 判断标准 |
|------|---------|
| 8-10 | 反常识/有冲突/容易引发讨论 |
| 4-7 | 有用但不会主动转发 |
| 0-3 | 太专业/太枯燥，只有极少数人感兴趣 |

## 三、选题等级制度

过滤和评分之后，选题进入以下等级体系：

### S级（旗舰选题）

- **评分：** 85分以上
- **特征：** 不可替代 + 高认知升级 + 刚好有时效锚点
- **投入：** 全力做，不限资源
- **节奏：** 占总选题 ~20%
- **标准：** 做完后用户会说"扬说这个系列太值了"

**范例：**
| 选题 | 评分 | 理由 |
|------|------|------|
| 《央行法修订：一部法律如何影响你的钱袋子》 | 92 | 全网没人系统解读+政策刚出台+专业纵深极大 |
| 《A股"421结构"：为什么4000点总也守不住》 | 88 | 独到分析框架+长期困扰散户的问题+中国独有语境 |
| 《英伟达VR200拆解：从一颗芯片到全球产业链震动》 | 87 | 热点事件+产业链拆解需要真功夫+可视化价值高 |
| 《美债收益率倒挂：一个指标如何预警经济衰退》 | 86 | 经典机制拆解+漫画因果链独特+长期搜索价值 |

### A级（核心选题，主力军）

- **评分：** 75-84分
- **特征：** 至少2个维度高分，其他维度不拖后腿
- **投入：** 标准资源投入
- **节奏：** 占总选题 ~50%
- **标准：** 做完后用户说"又学到了"

**范例：**
| 选题 | 评分 | 理由 |
|------|------|------|
| 《量子计算的20亿美元：投资逻辑还是政府补贴？》 | 82 | 热点+专业+但有类似分析 |
| 《全球央行为什么疯狂囤金》 | 80 | 好故事+数据丰富+可视化价值高 |
| 《A股"政策市"的历史与逻辑》 | 78 | 有角度+中国独有语境-但同类分析不少 |

### B级（候补选题）

- **评分：** 60-74分
- **特征：** 有用但不够独特，或角度需要再打磨
- **投入：** 放入选题池，暂不排期，等更好的角度出现
- **节奏：** 不主动推进，但保留
- **标准：** "这个选题不错，但还没找到非做不可的理由"

**随时可能被C级选题挤掉。**

### C级（直接淘汰）

- **评分：** 60分以下
- **特征：** 触碰了"三不碰"中的任意一条，或评分太低
- **动作：** 立即移入废弃池，标注原因

## 四、选题淘汰机制

### 4.1 每周淘汰制

每次新增选题的同时，必须淘汰同等数量的选题。

```
新增 1 个 → 淘汰 1 个（从B级里挑最弱的）
新增 3 个 → 淘汰 3 个
```

### 4.2 淘汰标准

优先淘汰以下类型：

1. **时效过期的** — 热点窗口已过
2. **被竞品抢先做的** — 别人做了且质量不差
3. **始终找不到"独到角度"的** — 打磨了2周还是一般
4. **选题池超过30个的** — 太多了说明筛选不够严格

### 4.3 季度大清洗

每季度末，对选题池做一次全面清理：
- S级/A级保留
- B级重新评分，低于65分的淘汰
- 所有"待排期超过3个月仍未启动"的选题自动降级

## 五、选题来源升级

### 5.1 来源优先级（重排）

```
第1优先：政策/法规解读
→ 央行动作、金融法规修订、宏观政策调整
→ 体现专业功底最强的类型

第2优先：市场机制拆解
→ 美债/汇率/利率/通胀/通缩的传导链条
→ 最能帮用户建立分析框架的类型

第3优先：产业链深度分析
→ AI硬件/新能源/芯片/存储的产业链逻辑
→ 可视化价值最高的类型

第4优先：热点事件解读
→ 财报/黑天鹅/地缘事件
→ 传播力最强的类型

第5优先：行为金融/投资心理
→ 散户行为/市场情绪/认知偏差
→ 最容易引发共鸣的类型
```

### 5.2 禁止来源

- ❌ 转载/翻译外网内容（无原创价值）
- ❌ 纯数据汇总类选题
- ❌ 个股分析/推荐类选题（合规风险）
- ❌ 已经被做烂的"经典选题"（如巴菲特、价值投资入门）

## 七、S-01 选题策划师升级版评估流程

```
输入: 候选选题（来源5.1中任意渠道）

Step 1: 三不碰过滤
  → 触碰任何一条 → 直接淘汰，标注原因
  → 全部通过 → 进入 Step 2

Step 2: 五维评分
  → 逐项评分，计算总分
  → 总分 < 60 → C级，淘汰
  → 60-74 → B级，放入候补池
  → 75-84 → A级，放入主选题池
  → ≥85 → S级，立即启动

Step 3: 竞品搜索验证
  → 搜索同类选题，确认角度是否独特
  → 如果已被竞品高质量覆盖 → 降一级或淘汰
  → 如果全网空白或质量都很差 → 加5分

Step 4: 一句话创新点
  → 必须写出能说服主编的"不同之处"
  → 写不出来 → 退回Step 2重新思考

Step 5: 主编复审
  → 提交评估卡 + 一句话创新点
  → 主编 3 票制: 全票通过 → 入池
                      2票通过 → 讨论
                      1票以下 → 淘汰
```

## 十、品控红线

以下情况之一发生时，**当期专题立即停止发布**：

1. **选题被更高品质的竞品抢先发布**（且对方质量不低于我们）
2. **核心数据在发布前发生变化**（如发布前两天市场反转）
3. **选题经不起"一句话说清楚创新点"这个测试**
4. **S-01 自己打了60分以下，但被强行推进**
5. **主编团队3人中2人反对**

## 附录：选题评估卡填写示例

### 正面示例：美债收益率倒挂

```
选题: 美债收益率倒挂的完整传导链：一个指标如何预警经济衰退

三不碰检查:
✅ 不是百度概念（百度只讲"什么是倒挂"，不讲完整传导链）
✅ 不是泛滥选题（大部分人讲概念，没有人用"传导链"角度）
✅ 不是事实罗列（核心是回答"为什么倒挂意味着衰退"）

五维评分:
```
