## 三、根因分析

### 3.1 系统性根因

经过梳理，本周全部10个问题可归因到以下**4个根因**：

```
┌─────────────────────────────────────────────────────────┐
│  根因A: 无自动校验门禁（P0-01, P0-02, P1-04, P2-05）      │
│  → 产出物缺乏硬性自动化验证                               │
│  → 人工审查覆盖率100%但正确率未达100%                     │
├─────────────────────────────────────────────────────────┤
│  根因B: 双轨渲染架构（P1-03, P2-07, P3-08）                │
│  → 预渲染HTML + JS动态渲染并行维护                        │
│  → 修改一处必须同步另一处,极易遗漏                        │
├─────────────────────────────────────────────────────────┤
│  根因C: 模板替换无防错（P0-02, P2-05, P3-09）              │
│  → 占位符`{{VAR}}`纯靠人工逐一手工替换                    │
│  → 占比符遗漏无任何提示                                  │
├─────────────────────────────────────────────────────────┤
│  根因D: 正则表达式脆弱（P2-06）                            │
│  → extract_series_tag对HTML结构变化敏感                   │
│  → 缺乏单元测试保护                                      │
└─────────────────────────────────────────────────────────┘
```

### 3.2 责任矩阵

| 根因 | 涉及Agent | 涉及Phase | 首次出现 |
|------|----------|-----------|---------|
| A. 无自动校验门禁 | C-07, C-08 | Phase 7, Phase 9 | Day 1 |
