## 一、注册前准备

### 1.1 注册主体选择

| 主体类型 | 所需材料 | 功能限制 | 推荐 |
|---------|---------|---------|------|
| **个人** | 身份证+微信号 | 无微信支付、无认证标识、部分功能受限 | ✅ **首推** |
| 企业 | 营业执照+对公账户 | 全部功能开放，可认证 | 后续升级 |

**建议**：先用个人主体注册启动，日活超过1万后再考虑企业认证迁移。

### 1.2 所需材料清单

```
□ 身份证（正反面清晰照片）
□ 本人实名微信号（已绑定银行卡）
□ 公众号名称（准备3-5个备选，避免重名）
    首选：小财漫画财经
    备选：漫话金融、小财夜报、每日财画
□ 头像文件（已准备 → docs/assets/avatar.svg → 需导出为PNG 300x300）
□ 功能介绍（30字以内，如：）
    "每天一张漫画，看懂财经大事"
□ 运营地区：中国大陆
□ 公众号类型：订阅号
```

## 二、注册步骤

### 2.1 注册流程

```
第1步：打开微信公众平台 → https://mp.weixin.qq.com/
第2步：点击"立即注册"
第3步：选择"订阅号"（不是服务号！）
第4步：填写邮箱 → 激活邮箱 → 设置密码
第5步：选择"个人"主体
第6步：填写身份证信息 → 管理员微信扫码验证
第7步：填写公众号信息
   - 名称：小财漫画财经
   - 功能介绍：每天一张漫画，看懂财经大事
   - 地区：按实际情况填写
第8步：提交审核
```

### 2.2 审核时间

| 环节 | 预计时间 | 备注 |
|------|---------|------|
| 邮箱激活 | 即时 | 检查垃圾邮箱 |
| 身份证验证 | 1-3分钟 | 扫码绑定管理员微信号 |
| 名称审核 | 1-7个工作日 | 如重名需更换 |
| **总计** | **1-7天** | 平均3天通过 |

### 2.3 名称审核注意事项

1. **避免敏感词**：不要包含"财经""投资""理财""股票""基金"等
2. **首选"小财漫画财经"**：若被拒，第二选择"漫话金融"
3. **商标风险**：注册前在商标查询系统检索是否被注册
4. **唯一性**：公众号名称全网唯一，先到先得

### 4.2 微信编辑器适配注意事项

```
1. 微信编辑器不支持 <style> 标签：
   → 需使用微信自带编辑器调整样式
   → 或将关键样式转为内联 style

2. 微信不支持 <audio> 标签：
   → 需在文章中插入"音频"卡片（编辑器中自带功能）
   → 或在上方引导回复关键词收听

3. SVG图片需要导出为PNG/JPG：
   → 微信不支持本地SVG渲染
   → 文章引用需用导出的图片文件

4. 推荐工作流：
   写文章 → 用微信编辑器排版 → 上传图片 → 插入音频 → 预览 → 发布
```

### 4.3 封面图规范

```
微信封面标准：
  一级封面（大图）：900 × 383px
  二级封面（小图）：383 × 383px（正方形）

## 六、常见问题

### Q1: 个人主体能发财经内容吗？
A: 可以。只要不涉及"荐股""操盘""代客理财"等违规内容，个人解读分析是完全允许的。

### Q2: 名字有"财经"审核会通过吗？
A: 建议避开。如果一定要用，选择"科技-科技资讯"类别比"财经"类别更容易过。

### Q3: 需要企业认证吗？
A: 前期不需要。认证主要好处：自定义菜单可放外链、获得搜索加权。1万粉丝前意义不大。

### Q4: 一天发几次？
A: 订阅号一天只能群发一次。建议早晚整合为一篇，或单发一篇。

### Q5: 如何变现？
A: 500粉丝可开通流量主（文末广告）。其他变现方式（付费阅读、知识星球、品牌合作）建议5万粉丝后再考虑。
