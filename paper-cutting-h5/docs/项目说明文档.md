# 《一纸千年：非遗剪纸文化互动 H5》项目说明

## 一、作品主题

本作品以中国非遗剪纸文化为主题，通过互动 H5 的形式展示剪纸的历史、纹样寓意、制作工艺、地域风格与当代传承。

## 二、设计理念

作品采用中国红（#B11226）、米白纸色（#F8EED8）和金色（#D8A24A）作为主色调，结合剪纸窗花、纸张纹理、传统纹样等视觉元素，营造浓厚的中国传统民俗氛围。所有页面统一红色剪纸主题，避免无关素材和现代风格混入。

## 三、主要功能

1. 全屏 H5 页面切换（支持滚动、触摸滑动、键盘方向键）
2. 剪纸历史时间轴（五大时代节点，点击查看详情）
3. 纹样寓意卡片 3D 翻转（六种常见剪纸纹样）
4. 剪纸工艺流程展示（六步骤点击高亮展开）
5. 拖拽拼窗花互动（Pointer Events 支持鼠标和触摸）
6. Canvas 擦除显影（擦除遮罩层显现剪纸图案）
7. 地域剪纸风格切换（五种地域风格介绍）
8. 剪纸知识问答（五道选择题 + 自动计分）
9. 剪纸祝福卡生成（输入祝福语 + 选择主题 + 生成卡片）
10. 保存祝福卡为图片（html2canvas 实现）
11. 传承保护页面（四大关键词折叠面板）
12. 结尾纸屑飘落动画 + 分享页

## 四、技术实现

- **前端技术**：HTML5 + CSS3 + JavaScript（原生，无框架依赖）
- **页面切换**：CSS transform translateY + transition 缓动动画
- **卡片翻转**：CSS 3D transform rotateY + perspective
- **拖拽拼图**：Pointer Events API 统一鼠标和触摸事件
- **擦除显影**：Canvas API globalCompositeOperation="destination-out"
- **图片保存**：html2canvas CDN 引入，截图 → toDataURL → 下载

## 五、素材说明

项目素材均围绕中国剪纸、非遗文化、传统纹样和纸张纹理进行设计。由于本项目为课程作业性质，视觉元素主要通过 CSS/SVG 代码生成（模拟剪纸镂空效果、纸张纹理等），减少对外部图片资源的依赖。如需替换为真实图片，代码中已预留明确位置。

## 六、作品亮点

- 传统文化主题突出，视觉风格高度统一
- 12 个页面覆盖"认知→体验→创作"完整闭环
- 交互形式丰富（拖拽、擦除、翻转、问答、生成）
- 手机端优先适配，支持触摸和鼠标双模式
- 祝福卡生成功能让用户有"作品产出感"
- 知识问答环节提升学习互动性
