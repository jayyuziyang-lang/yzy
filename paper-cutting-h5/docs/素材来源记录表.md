## 说明

本项目面向课程作业/学习展示场景。视觉元素（剪纸纹理、窗花图案、纸张背景等）主要通过 CSS 和 SVG 代码生成，而非使用外部图片资源。这样做的好处是：

1. 避免版权风险
2. 文件体积小，加载快速
3. 视觉风格完全可控
4. 所有效果均围绕剪纸主题设计

如需替换为真实剪纸图片，建议从以下渠道获取：
- 中国非物质文化遗产网 (https://www.ihchina.cn/)
- Wikimedia Commons 搜索 "Chinese paper cutting"
- Unsplash / Pexels 搜索 "Chinese paper cut"
