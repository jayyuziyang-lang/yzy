# 评分标准对照表

| 评分项 | 对应设计 | 提分点 |
|--------|---------|--------|
| 主题明确（20分） | 全部 12 个页面围绕非遗剪纸文化展开，无偏离 | 传统文化属性强，主题始终如一 |
| 内容完整（15分） | 覆盖历史、寓意、工艺、地域、传承五大维度 | 有文化深度，不是简单罗列 |
| 视觉美观（15分） | 中国红+米白纸色+金色配色，剪纸纹样装饰，纸张纹理背景 | 风格统一，氛围浓厚 |
| 交互丰富（15分） | 拖拽拼图、Canvas 擦除、卡片翻转、问答计分、祝福卡生成 | 10+ 个独立互动点 |
| 技术实现（15分） | Canvas API、Pointer Events、CSS 3D、html2canvas、Intersection Observer | 技术难度较高，多处使用原生 API |
| 用户体验（10分） | 手机端优先适配、按钮清晰、反馈明确、进度指示器 | 操作友好，支持触摸和鼠标 |
| 素材匹配（5分） | 视觉元素均为 CSS/SVG 生成的剪纸风格图案 | 不跑题，无不相关素材 |
| 文档规范（3分） | README、项目说明、素材来源表、评分对照表 | 提交完整 |
| 创新表达（2分） | 祝福卡生成+保存为图片、数字非遗体验 | 有创意亮点 |
| 完成度（加分） | 12 页完整闭环，从封面到结尾有始有终 | 作品像正式项目 |
