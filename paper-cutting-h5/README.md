# 一纸千年：非遗剪纸文化互动 H5

## 项目简介

这是一个以中国非遗剪纸文化为主题的互动 H5 作品。作品通过历史时间轴、纹样翻转、拖拽拼窗花、Canvas 擦除显影、知识问答和祝福卡生成等方式，让用户沉浸式了解中国剪纸文化。

## 运行方式

直接打开 `index.html` 即可运行。

如果使用本地服务器：

```bash
npx serve .
```

## 页面结构

| 序号 | 页面 | 说明 |
|------|------|------|
| 1 | 封面页 | 标题入场动画 + 开始体验 |
| 2 | 文化引入页 | 三张信息卡片淡入展示 |
| 3 | 历史时间轴页 | 五个历史节点点击查看详情 |
| 4 | 纹样寓意页 | 六张卡片 3D 翻转查看寓意 |
| 5 | 剪纸工艺页 | 六个工艺步骤点击高亮 |
| 6 | 拖拽拼窗花页 | 拖动碎片拼成完整窗花 |
| 7 | 擦除显影页 | Canvas 擦除遮罩显现剪纸 |
| 8 | 地域风格页 | 五种地域剪纸风格切换 |
| 9 | 知识问答页 | 五道选择题 + 计分反馈 |
| 10 | 祝福卡生成页 | 输入祝福语生成剪纸卡片 |
| 11 | 传承保护页 | 四大关键词折叠展开 |
| 12 | 结尾分享页 | 纸屑飘落 + 总结 + 返回 |

## 技术栈

- HTML5
- CSS3（CSS 变量 / 3D Transform / Animation）
- JavaScript（Pointer Events / Canvas API / Intersection Observer）
- html2canvas（祝福卡保存为图片）

## 作品亮点

- 传统文化主题突出，剪纸视觉风格统一
- 中国红 + 米白纸色 + 金色配色系统
- 12 个页面涵盖历史、工艺、地域、互动完整闭环
- 10+ 个互动点：拖拽、擦除、翻转、问答、卡片生成
- 支持桌面鼠标和手机触摸
- 可生成个性化剪纸祝福卡并保存为图片
