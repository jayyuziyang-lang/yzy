A股
4100股下跌 vs 存储芯片涨停潮——极致分化的一日

上证指数收报4057.78点，跌0.64%（-26.19点）；深证成指报15661.57点，跌0.27%；创业板指收4088.88点，跌0.83%。科创50逆势收涨0.69%，北证50跌1.28%。全市场约1339家上涨、4120家下跌，涨跌比约1:3。

成交量方面，沪深两市合计成交2.75万亿元，较前一交易日缩量约3726亿元，资金追高意愿不足。

领涨板块：电子化学品（+5.63%）、元件（+2.57%）、煤炭（+2.18%）。存储芯片概念爆发，德明利涨停（680.85元）、太极实业3天2板、兆易创新成交293亿元居A股第一。此外，中船系、超级电容、PCB概念午后拉升，京东方A涨停（市值超2000亿）。

领跌板块：油气开采（-3.98%）、工业金属（-3.82%）、白酒（-3.05%）。消费、零售、食品饮料全面走低，AI应用题材回调。

逻辑链
中东油价大涨 → 通胀预期升温 → 降息延后 → 消费/传统蓝筹估值承压 → 资金从消费/资源股撤出 → 涌入半导体/存储芯片（有独立景气逻辑）→ 上演极致分化
📌 投资视角

短期：存储芯片有独立于宏观的景气周期，涨价趋势贯穿全年，回调即是上车机会。但成交量萎缩意味着追高风险不小——2.75万亿的成交不足以支撑多主线并行。

关注：兆易创新（A股成交额第一，风向标）、德明利（涨停板情绪指标）、太极实业（3天2板的连续性）。

科技/AI
魏哲家：台积电数年内无法满足AI芯片需求，全年营收增超30%

6月4日，台积电在台湾新竹召开年度股东大会，董事长魏哲家释放了一系列强烈看多信号：

"台积电在全球的芯片供应在未来几年都无法满足AI所带动的需求。"
—— 魏哲家

四大核心观点：

1 营收高增：2026年全年营收以美元计算增速仍将超过30%，2025年合併营收3.8兆新台币，年增31.6%
2 AI供不应求：AI使用模式正从生成式AI加速向代理式AI与指令行动模式转变，推升token消耗量和算力需求，数年内产能无法满足
3 下一波增长：自动驾驶汽车和机器人将成为芯片业务的下一波长期增长动力
4 股价表现：台积电台股从2025年6月的950新台币涨至2425新台币，年增超155%；股利从18元调升至至少24元

全球半导体协会WSTS同步上调预测：2026年全球半导体市场规模预计突破1.5万亿美元，存储芯片同比增幅达惊人的249.5%，规模突破8000亿美元。TrendForce更将DRAM合约价从季增55-60%大幅上修至90-95%。

📌 投资视角

台积电释放的信号是整个AI产业链的"定心丸"——上游晶圆代工的产能瓶颈决定了AI军备竞赛的物理上限。存储芯片是本轮AI最确定的机会：涨价趋势贯穿全年已被机构共识确认。关注A股存储芯片产业链（兆易创新、佰维存储、德明利）及半导体设备材料（沪硅产业、华特气体）。

前沿/国际
中东一日三转：以黎有条件停火，美伊谈判卡壳，国会限战

中东局势在24小时内经历"升级→停火→不确定"的戏剧性转折：

【以黎停火：有条件协议，但有"空壳"风险】 美、以、黎三方联合声明，以色列与黎巴嫩同意实施停火。前提条件是：真主党必须完全停止开火并从利塔尼河以南撤出武装人员。但真主党明确表示"不承认谈判结果"——协议可能沦为一纸空文。

【美伊谈判僵局】 特朗普坚持将黎巴嫩冲突与美伊谈判"分开处理"，但伊朗坚持要求所有战线全面停火。特朗普私下对内塔尼亚胡"爆粗口"斥其"疯了"，美以裂痕公开化。此外，美众议院以215:208通过决议，限制特朗普对伊朗动武（除非获国会授权）。

地缘叙事线
伊朗袭击科威特/巴林 → 美军空袭格什姆岛 → 霍尔木兹海峡受阻 → 油价飙涨 → 以黎停火 → 油价暴跌 → 真主党拒绝 → 下一轮不确定
📌 投资视角

以黎停火对原油是短期利空（风险溢价回落），但对黄金未必是利空——若中东停火后美元走弱，反而可能支撑黄金。核心矛盾仍在美伊谈判，这是影响下半年能源价格的最大变量。

05
宏观
美联储褐皮书：通胀压力回升，市场从"何时降息"转向"何时加息"

6月3日发布的美联储褐皮书释放了明确信号：通胀正在重新抬头。

关键数据：4月PCE同比3.8%（2023年以来最大涨幅），核心PCE 3.3%，持续高于2%目标。中东冲突导致的能源成本飙升被褐皮书明确列为通胀压力的"首要来源"——部分企业反映燃料相关运输成本上涨超100%。能源涨价已外溢至航运、包装、食品、化肥等多个领域。

K型分化加剧：高收入家庭韧性较强，中等收入家庭趋于审慎，低收入家庭削减非必需品支出、依赖信用卡维持。制造业是亮点——9个辖区制造业以温和至强劲速度增长，国防和AI数据中心建设是主要支撑。

就业"低招聘、低裁员"僵局：11个辖区就业几乎无变化，员工因不确定性不愿跳槽，企业只招聘关键岗位。

政策预期反转：市场已从"何时降息"转向"何时加息"——6月会议98.4%概率维持不变，10月加息可能性高达50%，2027年3月前已完全消化一次加息。

情景推演

📈 路径A（加息路径·概率50%）： 中东局势持续紧张 → 油价维持高位 → 通胀继续走高 → 10月加息25bp → 全球风险资产承压

📉 路径B（维持路径·概率40%）： 中东停火实质化 → 油价回落 → 通胀企稳 → 美联储维持利率 → 市场情绪修复

极端路径C（概率10%）： 经济急剧放缓 → 滞胀 → 美联储陷入政策困境

商品/黄金
黄金破4500后回落，原油受停火消息冲击跳水4%

黄金：先抑后扬，一度突破4500美元

现货黄金全日走势呈现"V型反转"。亚市早盘承压于4448美元附近，日内一度下探4424美元（触及200日均线支撑）。欧盘时段企稳回升，最高触及4506.66美元（+1.63%），最终收报4495美元/盎司。

以黎停火消息出炉后，美元从两个月高位回落，为黄金提供支撑。但4500美元上方抛压明显，收盘未能站稳。

原油：以黎停火引发获利了结，美油暴跌4%

国际油价经历"先涨后崩"的剧烈波动。前一日（6月3日）WTI收报96.02美元（+2.4%），布伦特97.81美元（+1.9%）。以黎停火消息公布后，WTI直线跳水至92.32美元，布油跌至94.59美元，跌幅均超4%。

此前油价上涨逻辑包括：美伊冲突升级、霍尔木兹海峡通航受阻、美国原油库存连续八周下降。停火消息使部分风险溢价被快速挤出。

核心矛盾：黄金是利率预期资产，不是避险资产
以黎停火 → 风险偏好回升 → 美元走弱 → 黄金涨。油价跌 → 通胀预期回落 → 降息概率回升 → 黄金涨。停火对黄金→两面都利好，但4500是强阻力。
📌 投资视角

黄金：短期4500是多空分水岭。若中东持续降级→美元走弱→黄金有望突破；若通胀数据继续走高→加息预期压制→黄金回调至4400以下。长期看各国央行购金趋势不变。

原油：92美元是WTI的关键支撑。停火消息的冲击是短期的，美伊谈判才是中线变量。若谈判破裂，油价可能重回100美元以上。

📖 名词解释 · 小白友好版

术语大白话为什么重要
褐皮书（Beige Book）美联储每年发布8次的对全美经济形势的"体检报告"，由12个储备区分头撰写是美联储加息/降息决策的核心参考依据，褐皮书说通胀涨了，加息概率就上升
PCE物价指数美联储最看重的通胀指标，比CPI覆盖范围更广（包含企业、政府等更多支出）PCE 3.8%意味着通胀远高于2%目标，是降息的最大阻力
K型分化经济复苏"富人越富、穷人越穷"的"劈叉"走势，像字母K一样上下分叉说明经济不是真的复苏——只是富人更敢花钱，低收入群体靠信用卡度日
DRAM合约价存储芯片厂商和下游客户签订的批量采购价格，反映行业的真实供需季增90-95%意味着存储芯片供不应求的严重程度，直接决定相关公司利润
霍尔木兹海峡连接波斯湾和印度洋的狭窄水道，全球约1/3的石油贸易经过这里海峡一堵，油价就涨。这是中东地缘风险对全球油价影响的核心传导点
VIX恐慌指数衡量标普500期权隐含波动率的指标，市场越恐慌、VIX越高17.82虽不算极端恐慌，但单日涨8.7%说明市场情绪正在快速恶化
限制对伊动武决议美国众议院通过的决议，要求总统对伊朗动武前必须获得国会授权国会试图"拉住"特朗普，防止中东冲突进一步升级为全面战争
