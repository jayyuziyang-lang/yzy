6.2早报漫画：三重冲击波——NVIDIA PC芯片宣战、霍尔木兹海峡封锁、Anthropic近万亿美元IPO
🔥 头号焦点 — 英伟达GTC Taipei：RTX Spark芯片正式宣战PC处理器市场

6月1日，黄仁勋在GTC Taipei 2026大会上正式发布与微软联合研发的RTX Spark（N1X）超级芯片，宣告英伟达进军个人电脑处理器市场。 黄仁勋称"此次电脑重构的意义，堪比功能手机向智能手机的跨越式革新"。首批搭载设备覆盖微软、戴尔、惠普、华硕、联想、微星六大品牌，2026年秋季正式上市（来源：工商时报、搜狐科技）。

市场反应剧烈，芯片板块全线分化。 英伟达大涨6.3%，ARM暴涨16.1%（富国银行上调目标价至410美元），戴尔涨超10%，美光科技突破1000美元大关。英特尔暴跌4.7%，高通跌5.6%——英伟达一纸宣战，PC芯片格局一夜之间重新洗牌。AI应用软件股同步走强，ServiceNow涨9.2%、Cadence涨10.5%，市场焦点从基础设施向应用层扩散（来源：CNBC、网易财经）。

逻辑链：RTX Spark发布 → Arm PC生态成型 → x86（Intel/AMD）垄断出现第一道裂缝 → 微软/戴尔等巨头站队Arm → 端侧AI应用生态启动 → PC产业格局十年重塑。高盛重申英伟达买入评级，目标价285美元。本周关注博通财报（AI ASIC风向标）及美国非农就业报告。

⚡ 逻辑链
RTX Spark发布 → Arm PC生态成型 → x86垄断松动 → 巨头站队Arm → 端侧AI应用启动 → PC产业格局重塑（预计2-3年完成）
1A股6月开门"指数跌个股涨"，AI硬件切换软件

6月首个交易日（6/1），A股呈现鲜明的结构分化。上证指数微跌0.27%报4,057点，创业板指跌2.15%，科创50重挫5.00%。但全市场3,776只个股上涨，168只涨停——"指数跌、个股涨"背后，是一场大规模的风格切换。

电子板块单日净流出240亿元，半导体遭121亿抛售，东山精密、胜宏科技跌停。与此同时，计算机板块净流入41.6亿元居首，传媒板块净流入39.6亿元、大涨3.32%，AIGC概念全线爆发。资金正从AI硬件（算力基建）大规模迁移至AI应用（软件落地）——这不是撤退，是内部接力。两市成交2.88万亿元，较前日缩量约4,415亿（来源：证券时报、Wind）。

6月2日盘中观察：三大指数再度分化，创业板指涨超2%，上证微跌。AI PC概念、算力硬件领涨，但全市场近4,000只个股下跌——指数回来了，钱没回来。通信板块+2.68%领涨，传媒-3%领跌。

🔴 核心矛盾
科创50单日跌5% vs 全市场3,776只个股上涨——这不是同一件事。240亿从电子流出，40亿进了计算机——结构切换的规模在加速。关键是：这是短期轮动还是中期转向？取决于AI应用能否接住"从算力到落地"的叙事接力。
2美股三大指数齐创收盘新高，英伟达+6.3%引爆科技板块

美股周一（6/1）全线收涨，三大指数同步刷新历史收盘新高。道指涨0.09%报51,078点，纳指涨0.42%报27,086点，标普500涨0.26%报7,600点，标普科技板块单日飙升2.48%。

GTC Taipei大会成为全场焦点。英伟达RTX Spark芯片发布会带动整个芯片板块剧烈分化：ARM +16.1%（富国目标价410美元）、戴尔 +10%以上（摩根士丹利上调评级，目标价从170→448美元）、美光 +6.6%（首度突破1,000美元）、IBM +7.6%。英特尔-4.7%、高通-5.6%，传统PC芯片厂商一夜之间被打上了"旧时代"标签。

AI应用软件股同步走强，Cadence推出采用NVIDIA技术的AI芯片设计Agent，大涨10.5%；ServiceNow涨9.2%，IBM涨7.6%。市场焦点正从"AI基础设施"向"AI应用层"加速扩散——这是一条关键的信号线（来源：CNBC、各公司公告）。

⚡ 逻辑链
RTX Spark发布 → 市场重估PC芯片格局 → 直接受益者（ARM/存储/代工/AI应用）上涨 → 传统PC芯片（Intel/高通）承压 → AI应用扩散逻辑验证
3科技/AIAnthropic秘密提交IPO：估值9,650亿，超OpenAI

AI赛道再掀巨浪。Claude母公司Anthropic于6月1日向SEC秘密提交Form S-1上市申请，最新估值高达9,650亿美元，已超越OpenAI的8,520亿美元，成为全球估值最高的AI独角兽（来源：搜狐财经、新浪财经）。

核心数据：2026年5月年化收入运行率突破470亿美元，Q2预估营收109亿美元（同比增130%），有望首次实现季度盈利。80%收入来自企业客户（OpenAI仅约40%），企业粘性显著更强。高盛、摩根大通、摩根士丹利预计担任承销商，最快2026年秋季正式登陆资本市场。

与此同时，Alphabet（谷歌）6月1日宣布计划出售800亿美元股票，用于投资AI运算基础设施。叠加此前SpaceX以约1.75万亿美元估值筹备IPO，2026年被视为美股IPO市场空前盛大的一年——三家公司合计可能带来超过2,000亿美元的新增公开市场市值（来源：SEC文件、华尔街日报）。

🔴 关键判断
Anthropic的IPO意义不在于估值数字本身，而在于它标志着AI从"烧钱竞赛"进入"资本化阶段"。华尔街正在赌一件事：AI公司的收入增长能否追上甚至超越市场给予的估值溢价。Anthropic 80%企业客户占比vs OpenAI 40%——这不是50%的差距，是两种商业模式的分水岭。
4前沿/国际伊朗宣布计划彻底封锁霍尔木兹海峡，油价暴涨

中东局势骤然升级。伊朗革命卫队6月1日宣布，计划与"抵抗阵线"共同彻底封锁霍尔木兹海峡，并同步在曼德海峡开启行动。全球约20%的石油通过霍尔木兹海峡运输——每日约1,700万桶——这一通道若被切断，意味着全球能源供应的"心脏骤停"（来源：央视新闻、路透社）。

伊朗最高领袖军事顾问雷扎伊6月2日发出强硬警告："霍尔木兹海峡由伊朗控制，武装部队的耐心是有限的。"伊朗同时于6月1日宣布暂停与美国通过中间人的谈判，将黎巴嫩停火作为前提条件。革命卫队海军称已使用巡航导弹击中属于美以势力的船只，美军中央司令部称拦截了两枚瞄准驻科威特基地的弹道导弹（来源：国际在线、搜狐国际）。

市场反应剧烈。WTI原油暴涨5.49%至92.16美元/桶，布伦特涨4.24%至94.98美元/桶。特朗普称预计"未来一周内"与伊朗达成协议，同时下令大规模释放战略石油储备。这场能源博弈的走向，将是6月全球市场的核心变量。

🔮 情景推演

【路径A · 协议落地】 未来一周谈判达成 → 海峡复航 → 油价回落至$85-88 → 避险溢价全面退出 → 黄金短期承压

【路径B · 继续升级】（基准情景） 谈判拉锯 → 油价维持$90-100震荡 → 通胀预期再度升温 → 美联储加息预期强化 → 全球风险资产承压

5宏观中美各300亿对等降税，美方称关税"长期高于他国"

中国商务部6月1日确认，中美已就"贸易理事会"框架下各300亿美元及以上规模的双向对等降税达成原则共识。双方将成立政府间贸易理事会和投资理事会，将经贸磋商从"危机式应对"转向"机制化管理"（来源：商务部新闻发布会）。

中方承诺三件事：2026-2028年每年采购170亿美元美国农产品；恢复美牛肉及禽肉对华出口资格；按商业化原则重启引进200架波音飞机。美方将推进解决中方乳制品与水产品自动扣留问题，并就稀土供应链保持沟通。

然而，美国贸易代表格里尔同日表态：保住对华关税"相当棒"，对华关税将长期高于其他国家。 受降税预期刺激，远东至美西运价单周飙涨31.5%至4,149美元/FEU，达飞与赫伯罗特自6月1日起征收旺季附加费（来源：新华财经、雅虎财经）。

美国5月ISM制造业数据创四年新高，利率互换显示交易员已完全消化2027年3月前加息预期，美元指数走强。关税缓和的"蜜月期"与政策收紧的"紧箍咒"同时收紧——市场在双重信号间摇摆。

⚡ 宏观传导链
油价高企 + ISM制造业数据超预期 → 通胀预期升温 → 美联储加息预期强化 → 美元走强 → 成长股估值承压 → 黄金受压制。但同时：中美降税落地 → 贸易成本下降 → 航运/出口受益 → 部分对冲宏观压力。
6商品/黄金油价单日暴涨5.5%，金价高位回落近20%

大宗商品市场呈现罕见的冰火两重天。WTI原油7月合约暴涨5.49%至92.16美元/桶，布伦特8月合约涨4.24%至94.98美元/桶，均创近期最大单日涨幅。直接导火索是伊朗封锁霍尔木兹海峡的威胁——霍尔木兹每日承载约1,700万桶石油运输，占全球消费量的近20%。一旦被切断，亚洲石油进口国将首当其冲（来源：新华财经、国际金融报）。

黄金则上演"跷跷板"行情。现货金价跌至4,485美元/盎司，COMEX期货跌1.21%至4,515美元。自1月底创下近5,600美元历史高点后，金价已累计下跌近20%。油价暴涨推高通胀预期，反而强化了央行维持紧缩的预期，压制了黄金的避险吸引力。

伦铜涨1.4%至13,832美元/吨，受AI投资热潮及美国关税决定预期双向提振，5月累计涨幅约5%。基本金属普涨——伦锡+2.2%领涨、伦铝+1.4%、伦镍+1.0%。整个商品市场正被同一只手操纵：中东局势将油价推向高点，而通胀预期反过来压制金价与宽松预期。

🔴 核心信号
"石油-黄金-美元"三角正在剧烈重组。油价若持续站上95美元，通胀预期将进一步失控——届时受冲击的将不只是黄金，还有整个全球风险资产。这是当前市场最被低估的宏观风险。
🧠 今日逻辑框架：三条线，谁决定方向？
线一 · 英伟达PC芯片 + AI应用扩散

独立主线（不依赖停火/宏观） RTX Spark发布→芯片格局重组→AI应用扩散→科技板块结构性重估。这条线最"干净"——基本面驱动，不受地缘和政策左右。

关键信号：ARM+16%、戴尔+10%、ServiceNow+9%——资金正在从硬件向应用全面扩散。

线二 · 霍尔木兹局势 + 油价冲击

如果协议落地： 油价塌回$85→通胀预期降温→美联储压力减轻→成长股反弹→黄金短期承压

如果继续升级（基准）： 油价$90-100→通胀黏性维持→加息预期强化→风险偏好收紧→避险资产受益

线三 · Anthropic IPO + AI资本化

中长期叙事，独立于短周期波动。 Anthropic 9,650亿估值不是终点——如果下半年上市成功，它和SpaceX一起将推动美股IPO市场迎来历史性年度。这对AI行业的映射意义：从"烧钱竞赛"到"资本化阶段"，估值逻辑将从PS走向PE。

英伟达的宣战、伊朗的威胁、Anthropic的IPO——
三件事在同一天发生，各自独立，但指向同一个方向：
科技在重置、能源在博弈、AI在资本化。
站在中间的人，必须同时看懂这三条线。
