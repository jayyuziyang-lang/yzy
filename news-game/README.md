# 信息侦探：真假新闻挑战

## 项目简介

一款侦探主题的新闻素养互动 H5 小游戏。用户扮演"信息侦探"，在限时压力下判断 5 条新闻线索的真伪，通过连击系统获取额外加分，最终获得个人"新闻辨别力评分"和专属二维码分享卡。

## 游戏特色

- **侦探主题视觉**：深色侦探风格 UI，CSS 手绘侦探角色，粒子背景
- **限时挑战**：每题 15 秒倒计时，超时自动判错，环形计时器动画
- **连击加分系统**：连续答对触发连击（2连击+5分，3连击+10分，4+连击+15分）
- **分数动画**：结果页分数环动画 + 数字滚动效果
- **撒花庆祝**：80 分以上触发全屏撒花特效
- **动态二维码**：自动根据部署地址生成二维码，扫码即可体验

## 游戏玩法

1. 阅读案情简介，了解调查任务
2. 学习侦探行动守则
3. 在 15 秒内判断每条新闻线索：**可信** 还是 **存疑**
4. 连续答对触发连击加分
5. 查看调查结论和详细统计
6. 扫码或复制链接分享你的侦探战绩

## 本地运行

直接用浏览器打开 `index.html` 即可体验。

或使用本地服务器：

```bash
cd news-game
python -m http.server 8080
# 打开 http://localhost:8080
```

## 部署方法

### 方案一：GitHub Pages（推荐 · 免费）

1. 新建 GitHub 仓库（如 `news-game`）
2. 上传全部项目文件
3. Settings → Pages → Source: `Deploy from a branch` → `main` → `/ (root)`
4. 保存后等待部署完成
5. 最终链接：`https://你的用户名.github.io/news-game/`

### 方案二：Vercel

1. 登录 [Vercel](https://vercel.com) → Add New Project
2. 导入 GitHub 仓库 → Deploy
3. 获得链接：`https://news-game.vercel.app/`

### 方案三：Netlify

1. 登录 [Netlify](https://netlify.com)
2. 拖入项目文件夹 → 自动部署

## 二维码

项目使用 QR Server API 自动生成二维码。部署后，分享页会自动显示指向部署地址的二维码。

如果 API 不可用，可手动使用 [草料二维码](https://cli.im/) 生成。
