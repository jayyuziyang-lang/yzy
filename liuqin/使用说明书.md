# 柳琴戏非遗 H5 · 使用说明书

## 项目概述

基于 Swiper.js 的竖屏滑动 H5，共 10 页，包含 9 道柳琴戏知识选择题。答题后弹窗即时反馈对错，答完全部题目后生成个性化成绩海报（含真实二维码），可保存到相册分享。

- **主题：** 一缕拉魂腔，沂蒙柳琴魂
- **内容：** 国家级非遗临沂柳琴戏知识闯关
- **适用场景：** 微信公众号、朋友圈分享、非遗文化宣传活动

---

## 快速体验

```
1. 确保电脑联网（需加载 CDN 资源）
2. 双击 index.html
3. 浏览器打开即可体验
```

推荐使用 Chrome / Edge / Safari。手机端可部署到服务器后用微信扫码访问。

## 操作说明

### 答题流程
- 点击选项 → 弹窗显示对错并揭晓正确答案 → 底部红色按钮亮起 → 点击进入下一页
- 未答完题目前按钮为灰色不可点击，防止跳题

### 音频系统
- 右下角圆形按钮点击开始播放音频链：**开头导入 → 导入2 → 柳琴戏唱段（完整）→ 循环回开头**
- 三段音频无缝衔接，无空白、无重叠，播完自动循环
- 再次点击圆形按钮可暂停/恢复音频
- 打开视频时自动暂停背景音频，关闭视频后自动恢复

### 视频按钮
- 每页有音频按钮（如"《姐儿妞》片段·柳琴戏母体曲调"），点击后底部弹出 Bilibili 播放器
- 再次点击同一按钮关闭播放器
- 打开视频时自动暂停背景音乐，关闭视频后自动恢复
- 如 iframe 5 秒内未加载成功，自动在新窗口打开 Bilibili 页面

### 海报生成与分享
- 第 10 页自动渲染成绩海报预览
- 点击"保存成绩海报"→ 生成高清海报（750×1000px，含真实二维码）→ 弹窗显示
- **手机端：** 长按海报图片 → 保存到相册
- **电脑端：** 右键海报图片 → 另存为
- 海报包含：分数、星级、评语、标语、可扫描二维码

## 如何部署

### 方式一：本地预览
双击 `index.html`，浏览器打开即可。需联网加载 CDN 资源。

### 方式二：部署到服务器
整个 `柳琴/` 文件夹上传到任意静态服务器即可，**无需后端、无需数据库、无需构建工具**：
- **GitHub Pages：** push → Settings → Pages → 选择分支保存
- **Vercel / Netlify：** 拖拽文件夹部署
- **腾讯云 COS / 阿里云 OSS：** 上传 → 开启静态网站托管

### 方式三：嵌入微信公众号
部署后获得 URL → 公众号后台 → 新建图文 → 原文链接填入 URL。

### 二维码说明
海报中的二维码编码内容为 `https://liuqin.vercel.app`。部署后需修改二维码链接：

1. 打开 `js/app.js`
2. 搜索 `https://liuqin.vercel.app`（共 1 处）
3. 替换为你的实际部署 URL
4. 保存即可

## 如何修改内容

### 修改题目和文案
编辑 `js/quiz-data.js`，以封面题为例：

```javascript
cover: {
  quiz: {
    question: '柳琴戏俗称什么名字？',     // 题目
    options: ['梆子腔', '拉魂腔', '茂腔', '吕剧'],  // 四个选项
    correct: 1,                          // 正确答案索引（0=第一个）
    wrongMsg: '答错啦！...',              // 答错弹窗文案
    correctMsg: '回答正确！...',          // 答对弹窗文案
  },
}
```

### 修改得分评语
在 `js/quiz-data.js` 底部 `scoreMessages` 对象中修改。

### 修改配色
在 `css/style.css` 顶部 `:root` 块中修改 CSS 变量即可全局换色：

```css
:root {
  --primary: #9B2D30;    /* 主题色（按钮、标签） */
  --gold: #B8860B;       /* 金色（进度条、得分环） */
  --bg: #FDF8F2;         /* 页面背景色 */
  --text: #2C2416;       /* 正文颜色 */
}
```

图片上叠加文字的颜色在 `css/style.css` 的 `.vo-label`（金色）、`.vo-sub`（浅金）、`.vo-tag`（红底白字）等类中修改。

### 替换配图
将新图片放入 `images/` 目录，覆盖对应文件即可。图片文件名对应关系见上方"文件结构"。

### 替换音频
将新 MP3 放入 `images/` 目录，覆盖对应文件：
- `opening.mp3` — 第一段（开头导入）
- `intro2.mp3` — 第二段（导入2）
- `sing.mp3` — 第三段（柳琴戏唱段，播完自动循环回第一段）

三段播完后自动无缝循环，无需修改代码。

### 修改 Bilibili 视频
编辑 `js/quiz-data.js`，每页有 `bilibili` 字段（填入 BVID），对应页面的音频按钮会加载该视频。

## 技术栈

| 组件 | 技术 | 加载方式 |
|------|------|----------|
| 竖屏滑动 | Swiper.js v11 | CDN |
| 二维码 | QRCode.js v1.0 | CDN |
| 海报生成 | 原生 Canvas API | 浏览器内置 |
| 视频 | Bilibili iframe 嵌入 | 在线 |

**无框架、无构建工具、无后端依赖。** 仅 Swiper.js 和 QRCode.js 通过 CDN 加载，其余全部本地运行。

## 常见问题

**Q: 双击 index.html 样式异常或无法滑动？**
A: 需联网加载 Swiper.js CDN。检查网络连接后刷新页面。

**Q: 海报生成失败？**
A: 海报使用浏览器原生 Canvas API 生成，无需额外 CDN。确认 `images/poster_bg.jpg` 存在。如仍失败，检查浏览器控制台错误信息。

**Q: 音频不播放？**
A: 浏览器可能拦截自动播放。点击右下角音频按钮即可开始播放。确保 `images/` 目录下三个 MP3 文件完整。

**Q: Bilibili 播放器打不开？**
A: 系统浏览器（Chrome/Edge/Safari）正常。微信内置浏览器可能拦截 iframe，会自动降级为新窗口打开 Bilibili 页面。

**Q: 二维码扫描无效？**
A: 二维码编码的是 `https://liuqin.vercel.app`（占位 URL）。部署后需在 `js/app.js` 中替换为你的实际网址。

**Q: 如何禁止用户跳过题目？**
A: 当前已实现：答完当前页题目之前，向下滑动被锁定，导航按钮为灰色不可点击。无需额外设置。
